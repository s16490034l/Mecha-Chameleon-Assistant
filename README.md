## 🔧 Technical Details

The Camouflage Breaker works by:

1. Reading player position data from the game's memory.
2. Analyzing the color palette of the player's camouflage texture.
3. Comparing it with the background colors at their current position.
4. Calculating a camouflage quality score — lower score = more visible.
5. Highlighting the player if their camouflage score is below the threshold.

This algorithm can be adjusted through the settings menu (default threshold: 60%).
